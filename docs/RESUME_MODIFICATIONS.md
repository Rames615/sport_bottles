# Résumé fonctionnel des évolutions

## Objectif

Fournir une synthèse courte des principales évolutions visibles côté utilisateur.

## Évolutions majeures

- catalogue produit clarifié ;
- fiche produit dédiée ;
- panier persistant et plus robuste ;
- tunnel de commande en plusieurs étapes ;
- paiement Stripe confirmé par webhook ;
- espace utilisateur avec profil et historique de commandes ;
- administration structurée via EasyAdmin ;
- amélioration des pages légales et du consentement cookies.

## Positionnement

Ce document résume les changements du point de vue fonctionnel. Pour le détail technique, voir `docs/RESUME_CODE_MODIFICATIONS.md`.
