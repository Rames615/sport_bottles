# Guide rapide - catalogue produit

## Objectif

Ce document explique comment intervenir rapidement sur le module produit sans relire toute la documentation détaillée.

## Ajouter ou modifier un produit

Un produit repose sur les champs suivants :

- désignation ;
- description ;
- prix ;
- stock ;
- image (`imgPath`) ;
- capacité ;
- température éventuelle ;
- catégorie.

## Points pratiques

- les images référencées par `imgPath` doivent exister dans le dossier public prévu par l'interface ;
- une catégorie doit être liée au produit pour un affichage cohérent dans le catalogue ;
- les promotions actives influencent le prix final affiché.

## Vérifier après modification

- affichage dans le catalogue ;
- affichage dans la bonne catégorie ;
- affichage de la fiche détail ;
- ajout au panier ;
- cohérence du prix et du stock.
