# Hero section - cadrage fonctionnel et visuel

## Objectif

Ce document fixe le rôle de la hero section sur la page d'accueil.

## Rôle

La hero section doit :

- présenter immédiatement la promesse de marque ;
- installer l'univers visuel du site ;
- orienter l'utilisateur vers le catalogue ou les meilleures ventes ;
- soutenir la perception d'un site e-commerce clair et crédible.

## Messages clés à porter

- durabilité ;
- mode de vie actif ;
- simplicité ;
- responsabilité environnementale.

## Principes de composition

- un titre fort et lisible ;
- un sous-titre court de réassurance ;
- un visuel principal contextualisé ;
- un appel à l'action prioritaire et un secondaire.

## Règle d'alignement avec le reste du site

La hero section ne doit pas fonctionner comme un bloc isolé. Sa palette, son ton et ses appels à l'action doivent rester cohérents avec :

- le catalogue produit ;
- les promotions ;
- la navigation principale.
