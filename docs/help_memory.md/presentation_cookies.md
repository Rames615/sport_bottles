# Présentation - consentement aux cookies

## Objectif

Présenter de façon concise le module de consentement aux cookies ajouté au projet.

## Fonctionnement

- un composant de consentement s'affiche lors de la première visite ;
- l'utilisateur peut accepter ou refuser ;
- le choix est mémorisé localement ;
- le composant n'est plus réaffiché tant que la préférence est présente.

## Intérêt dans le projet

- amélioration de la transparence ;
- meilleure conformité vis-à-vis des bonnes pratiques de protection des données ;
- complément cohérent aux pages légales du site.

## Point d'attention

Le stockage local du choix répond au besoin UX. Pour une traçabilité réglementaire plus forte, une persistance côté serveur pourrait être envisagée dans une version ultérieure.
