# Historique synthétique des modifications produit

## Objectif

Conserver une trace courte des grandes évolutions du module produit, sans multiplier les détails déjà décrits ailleurs.

## Évolutions notables

- amélioration progressive des cartes produit ;
- clarification des zones d'action ;
- réduction de la dépendance à des interactions complexes de type modale ;
- mise en place d'une page détail dédiée ;
- meilleure cohérence entre catalogue, promotions et ajout au panier.

## Positionnement du document

Ce fichier conserve la mémoire des évolutions. Pour l'état courant, consulter prioritairement :

- `docs/PRODUCT_ARCHITECTURE.md` ;
- `docs/PRODUCT_IMPLEMENTATION_SUMMARY.md` ;
- `docs/productDetail.md`.
